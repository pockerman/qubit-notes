# qubit-note: MQTT 101

## Overview

We'll introduce MQTT, which stands for Message Queue Telemetry Transport, a system that
can connect different programs running and sending/receiving messages with each other,
either on the robot or from other devices, like a smartphone. One program or service can deal
with motor topics.


We'll introduce MQTT, which stands for Message Queue Telemetry Transport, a system that
can connect different programs running and sending/receiving messages with each other,
either on the robot or from other devices, like a smartphone. One program or service can deal
with motor topics.

Mosquitto needs a username and password for our code to connect with it. We've stored the
state of the package install in a variable called mosquitto_packages, and if that has changed,
we use a command to set the password. We need to ensure that Mosquitto owns this password
file.

**Remark:**

The tools MQTT UI (https://github.com/EdJoPaTo/mqttui) and MQTT Explorer
